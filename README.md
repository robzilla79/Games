# Neon Slam Wrestling RPG 2.0 - Living Ring Update

A self-contained mobile-friendly browser wrestling RPG. Create a custom rookie, train, climb fictional GCE rankings, chase belts, trigger storylines, and watch animated matches unfold in a Canvas arena.

## New in 2.0

- Animated match viewer with ring, crowd, ropes, impact bursts, banners, and wrestler sprites
- Autonomous wrestler AI: wrestlers make their own in-match decisions
- Style/personality-driven behavior for Powerhouse, High-Flyer, Technician, Brawler, Showman, Monster, Underdog, and Heel Mastermind
- Strategy presets before matches: Balanced, Play It Safe, Go All Out, Work the Crowd, Target Weakness, Hunt the Finish, and Cheat to Win
- Match actions: strikes, grapples, aerial attacks, counters, submissions, dirty tactics, pins, comeback sequences, and finishers
- Live HUD showing health, stamina, momentum, and current action
- Match speed controls and skip-to-result
- Career integration: wins/losses affect reputation, money, fans, morale, stamina, rankings, and belts
- iPhone-friendly layout, installable web app manifest, and offline service worker when hosted

## How to play on iPhone

1. Upload this folder to GitHub Pages, Netlify, Cloudflare Pages, or any static host.
2. Open `index.html` in Safari.
3. Tap Share -> Add to Home Screen.
4. Launch it like an app.

Progress saves in browser storage.
